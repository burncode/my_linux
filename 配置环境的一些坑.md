# 光盘源码中需要进行的一些修改

[VMware配置Ubuntu](https://blog.csdn.net/weixin_42249196/article/details/107996543)

安装bochs参考 安装环境.docx

还需要安装nasm:

```
解压 nasm-2.15.03.zip
进入nasm-2.15.03目录
chmod +x nasm-2.15.03
执行./configure
make
sudo make install
```



拷贝光盘源码到/root/bochs/bochs-2.6.9/下

如直接使用本仓库的源码,则可不用进行以下修改,可直接到运行处

## MakeFile文件修改
CC		= gcc -m32 -std=c99
		LD		= ld -m elf_i386

## bochsrc文件修改
romimage: file=\$BXSHARE/BIOS-bochs-latest
		vgaromimage: file=\$BXSHARE/VGABIOS-lgpl-latest

keyboard: keymap=$BXSHARE/keymaps/x11-pc-de.map

## 找不到80m.img

gzip -d 80m.img.gz

如果解压后不正当退出,也会导致找不到80m.img,此时把源码目录下80.img.lock文件删除即可

# 运行
make  image

bochs -f bochsrc
